Introduction

Welcome to the project documentation. This project is built using Java, JavaScript, Spring Boot, and Maven.

Setup

Instructions on how to set up the project locally.


git clone https://github.com/kurojs/McView3D.git
cd project
mvn clean install

Usage

To run the project, use the following command:


mvn spring-boot:run

Open your browser and navigate to http://localhost:8080 to see the application in action.

Parameters

The application supports the following URL parameters:

  • skin - The name of the Minecraft skin to display (default: kuromy888).
  • height - The height of the canvas (default: 400).
  • width - The width of the canvas (default: 400).
  • cape - The URL of the cape to display. If set to default, it will try to load the cape from capes.dev.

Examples

Example URL to display a custom skin:


http://localhost:8080?skin=customSkinName

Example URL to display a custom skin with specific canvas dimensions:


http://localhost:8080?skin=customSkinName&height=500&width=500

Example URL to display a custom skin with a specific cape:


http://localhost:8080?skin=customSkinName&cape=customCapeUrl
